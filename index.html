CROWD DENSITY ESTIMATOR

This is a desktop based application which can detect the density of crowd in public area.
After the covid-19 breakdown, a massive awarness of social distancing is noticable. This application can contribute a lot in this.
A Multi column Convolution Neural network (MCNN) is used to detect different size of heads in a crowd. As an additional feature system will generate an alarm whenever crowd in a particular place is higher than certain threshold. Extreamly helpful for crowd management.
